# PLC Conveyor Control Project

## Status
✅ Working – 6 networks implemented

## Quick Info
- **PLC:** Siemens S7-1200 DC/DC/DC
- **Language:** Ladder Logic
- **Difficulty:** Intermediate

## Features
- Emergency stop
- Belt misalignment detection with manual reset
- Batch counting Mode 1 (10 products → 10 sec dwell)
- Batch counting Mode 2 (20 products → 10 sec dwell)
- Extended batch tracking (20 products)
- Auto/Manual mode
- Starvation timer (stops if no product for 10 sec)
